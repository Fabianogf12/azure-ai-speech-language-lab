# Laboratório - Azure Speech Studio e Language Studio

## Objetivo do Projeto
Este repositório tem como objetivo documentar a prática realizada com os serviços **Azure Speech Studio** e **Language Studio**, aplicando conceitos de **Inteligência Artificial (IA)** voltados para fala e linguagem natural.  
O projeto faz parte do curso **"Seus Primeiros Passos com IA"** da **Digital Innovation One (DIO)** em parceria com a **Microsoft**.

---

## Azure Speech Studio

### Acesso ao portal
https://speech.microsoft.com/portal

### Conceito
O **Speech Studio** é uma plataforma do Azure que permite criar, testar e ajustar modelos de reconhecimento e síntese de fala.  
Com ele, é possível transformar fala em texto (**Speech-to-Text**) e texto em fala (**Text-to-Speech**) com vozes neurais em diversos idiomas.

### Etapas realizadas
1. Acessar o portal do **Speech Studio** com a conta Microsoft.  
2. Escolher o recurso **Text to Speech (Texto para Fala)**.  
3. No campo de texto, digitar uma frase de exemplo:  
   "Olá, este é um teste do serviço de fala do Microsoft Azure."  
4. Selecionar o idioma **Português (Brazil)** e a voz **Francisca (pt-BR, Neural)**.  
5. Executar a conversão de texto em fala e ouvir o resultado.  

### Resultado
O sistema converteu o texto em voz natural, demonstrando a capacidade de gerar fala humanizada a partir de texto.  
Esse recurso pode ser aplicado em assistentes virtuais, bots de atendimento e leitores automáticos de texto.

## Azure Language Studio

### Acesso ao portal
https://language.cognitive.azure.com/

### Conceito
O **Language Studio** é uma ferramenta do Azure voltada para o processamento de linguagem natural (**NLP**).  
Com ela, é possível realizar tarefas como **análise de sentimentos**, **extração de frases-chave** e **detecção de idioma** em textos.

### Etapas realizadas
1. Acessar o portal do **Language Studio** com a conta Microsoft.

## Conclusão
Este laboratório permitiu compreender como os serviços de Inteligência Artificial da Microsoft Azure podem ser utilizados para processar fala e linguagem natural.  
O **Speech Studio** mostrou como converter texto em fala humanizada, enquanto o **Language Studio** demonstrou a capacidade de analisar sentimentos e extrair insights de textos.  
Esses recursos são fundamentais para aplicações modernas em chatbots, assistentes virtuais e análise de feedbacks de clientes.
